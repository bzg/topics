#+title: Topics

Topics generates a static HTML/CSS/JS site from topics data.

#+html: <img src="topics.webp" />

* Install

Assuming [[https://github.com/babashka/bbin][bbin]] is installed:

#+begin_src sh
bbin install io.github.bzg/topics
#+end_src

* Run

Suppose you have an =input.json= file structured like this:

#+begin_src json
[
  {
    "title": "Topic title",
    "content": "<p>HTML</p>",
    "category": "Category name"
  }
]
#+end_src

To output an =index.html= file:

#+begin_src sh
topics input.json
#+end_src

* Customize

You can add configuration in a =config.edn= file:

#+begin_src clojure
 {:title   "My FAQ"
  :tagline "Frequently asked questions"
  :footer  "<a href=\"https://example.com\">My site</a>"
  :source  "https://example.com/data/faq.json"
  :theme   "swh"}
#+end_src

Here are the configuration keys:

: :title     - Website title (default: "Topics")
: :tagline   - Website tagline (default: "Topics to explore")
: :footer    - Footer HTML
: :source    - URL displayed as content source
: :css-theme - CSS theme (see below)

Use =$ topics -c my_config.edn input.json= to use another file name than
=config.edn=.

The =-C= / =--css-theme= option accepts four forms:

- *https:// URL*: linked as an external stylesheet via =<link>=
- *file:/// URL*: the local file content is inlined in =<style>=
- *Local .css path* (relative or absolute): the file content is inlined in =<style>=
- *Theme name* (e.g. =doric=, =org=, =swh=): resolved to a [[https://github.com/bzg/pico-themes][pico-themes]] CDN URL

#+begin_src sh
$ topics -C swh input.json
$ topics -C https://example.com/my-theme.css input.json
$ topics -C file:///home/user/my-theme.css input.json
$ topics -C ./my-theme.css input.json
#+end_src
